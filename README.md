# Rock Paper Scissors Game 🎮

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. The game allows users to play against the computer, which generates random choices each round. It provides instant feedback on the result and keeps track of the player's score through a clean and responsive interface.

## 🚀 Features

- Play Rock, Paper, Scissors against the computer
- Random computer move generation
- Real-time game result display (Win, Lose, or Draw)
- Live score tracking
- Responsive and user-friendly interface
- Clean and organized code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
